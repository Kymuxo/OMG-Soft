Known Issues

The following is a list of known issues:

  • The maximum image size at present is 8000 x 8000
  • Pixlr may not run if your machine doesn't have DirectX or has a version below 11

Recommended minimum system requirements

To ensure Autodesk Pixlr runs to its potential, please ensure that your hardware meets the following requirements:

  • Windows®7 (32-bit, 64-bit) and Windows®8 (32-bit, 64-bit)
  • 1 GHz CPU or faster Intel or AMD processor and 1GB of memory (32-bit) or 2GB of memory (64-bit)
  • Video card and 128 MB video memory support with DirectX (version 11 or higher) support
  • 450 MB hard disk space
  • Mouse, touchpad, or touch screen
  • 1024 x 768 Screen Resolution
